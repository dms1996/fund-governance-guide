# BA | Codex

A comprehensive, bilingual (EN/PT) reference guide for Business Analysis in the Fund Governance industry.

## Overview

This is a personal knowledge base covering the full spectrum of fund governance operations, designed as an interactive HTML document. It serves as both a study resource and a professional reference for Business Analysts working in investment fund services.

## Topics Covered

- **Fund Governance** -- Industry structure, Management Companies, AIFMs
- **Investment Funds** -- UCITS and Alternative Investment Funds (AIFs)
- **Regulatory Framework** -- EU, Luxembourg, Ireland, UK jurisdictions
- **ManCo Operations** -- Organisational structure, oversight, delegation, NAV, reporting
- **Risk Management & Compliance** -- Frameworks, risk types, regulatory compliance
- **Business Analysis** -- Role, methodologies, tools, deliverables, stakeholder management
- **Financial Analysis** -- Performance metrics, fee structures, KPIs, MI Packs
- **Technology & Data** -- Platforms, data governance, automation, RegTech
- **Professional Development** -- Certifications (CBAP, CFA, FRM), career paths
- **Integrated Case Study** -- End-to-end practical scenario

## Features

- Bilingual toggle (English / Portuguese)
- Light and dark mode
- Full-text search
- Sidebar navigation with scroll tracking
- Print-optimised PDF output with cover page and table of contents
- Responsive design (desktop and mobile)

## Usage

Open `index.html` in any modern browser. No dependencies, no build step, no server required.
